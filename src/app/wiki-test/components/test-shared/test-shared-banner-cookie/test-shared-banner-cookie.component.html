Banner Cookie

Lo stile del banner dei cookie è definito in styles.scss
Posizione TOP | Posizione BOTTOM | Posizione RIGHT | Posizione LEFT 
    

      
      Lo stile del banner dei cookie è definito dentro `styles.scss` e si può modificare o sovrascivere in base alle esigenze

      ```css
        .banner-cookie {
          background-color: #6c757d !important;
        
          .close{
            color: darkred !important;   
          }
          .text {
            text-align: center;
            /* top or bottom */
            .more-info {
              float: right;
              margin-right: 1rem;
            }
            .accept{
              float: right;
              margin-right: 1rem;
            }
          }
          .more-info {
            text-align:center;
          }
          .accept{
            text-align:center;
          }
        }
      ```

      L'utilizzo è registrato in `app.component.html`

      ```html
        ...
        <ddc-init-banner-cookie [text]="'APP.COOKIE.INFO'"></ddc-init-banner-cookie>
        ...
      ```

      Per modificare lo stile dei pulsanti "Leggi" e "Accetta" bisogna passare i parametri **moreInfoClass** o **moreInfoStyle** e **acceptClass** o **acceptStyle**

Address Search

Componente per la ricerca di indirizzi.
Parametri di Input :
  • limit: number - numero massimo di record (default 5)
  • digits: number - numero massimo di caratteri digitati prima della ricerca (default 3)
  • extratags: boolean - Aggiunta di informazioni per indirizzo (default true)
  • field : FormFieldModel - Campo di input associato
  • flgMore: boolean - Indica se Ã¨ necessario mostrare il pulsante more per risultati multipli
Parametri di Output :
  • addressEmit: EventEmitter<OpenstreetLocationModel> - numero massimo di record (default 5)

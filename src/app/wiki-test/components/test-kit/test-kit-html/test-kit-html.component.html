Il componente ProgressBarComponent consente di eseguire una barra di progressione percentuale e viene configuarto con i seguenti parametri di input
  • backgroundColor che definisce il colore dello sfondo vuoto della barra di avanzamento
  • color che definisce il colore proprio della barra di avanzamento
  • height che definisce lo spessore in altezza della barra
      


        ```html
          <a class="btn btn-xs btn-primary mr-2" (click)="progressBar1.start()">Start</a>
          <a class="btn btn-xs btn-danger mr-2" (click)="progressBar1.stop()">Stop</a>

          <ddc-init-progress-bar #progressBar1 [cssClass]="'mt-2'" [backgroundColor]="'darkgray'" [color]="'darkblue'"
            [height]="'16px'" [intervalGrow]="0.05" [showText]="true">
          </ddc-init-progress-bar>

        ```

La direttiva DdcCssVarDirective consente di assegnare proprietà di stile con valore una variabile css e viene configuarto con i seguenti parametri di input
  • cssVars Array di oggetti key,value dove key è la proprietà css e value è la variabile da assegnare
      


        ```html
          <div ddcCssVar [cssVars]="[{key:'background', value:'dark'}, {key:'color', value:'light'}]">Text for div</div>

        ```

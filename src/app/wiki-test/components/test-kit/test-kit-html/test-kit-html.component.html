Il componente ProgressBarComponent consente di eseguire una barra di progressione percentuale e viene configuarto con i seguenti parametri di input
  • backgroundColor che definisce il colore dello sfondo vuoto della barra di avanzamento
  • color che definisce il colore proprio della barra di avanzamento
  • height che definisce lo spessore in altezza della barra
  • size dimensioni senza specificare il size ('xs' | 'sm' | 'md' | 'lg' | 'xlg')
  • rounded se true mostra la barra con i bordi arrotondati
  • showText se true mostra il testo di avanzamento percentuale
  • hideOnComplete se true nasconde la progress al termine
  • showOnStart se true visualizza la progress dopo l'avvio (necessita di un pulsante o function che fa partire il loading)
  • intervalGrow in secondi indica il tempo di avanzamento del loader
  • timeoutHide in secondi indica il tempo di spegnimento del loader
      


        ```html
          <a class="btn btn-xs btn-primary mr-2" (click)="progressBar1.start()">Start</a>
          <a class="btn btn-xs btn-danger mr-2" (click)="progressBar1.stop()">Stop</a>

          <ddc-init-progress-bar #progressBar1 [cssClass]="'mt-2'" [backgroundColor]="'darkgray'" [color]="'darkblue'"
            [height]="'16px'" [intervalGrow]="0.05" [showText]="true">
          </ddc-init-progress-bar>

        ```

Il componente ProgressCircleComponent consente di eseguire una cerchio di progressione percentuale e viene configuarto con i seguenti parametri di input
  • backgroundColor che definisce il colore dello sfondo vuoto del cerchio di avanzamento
  • borderColor che definisce il colore del bordo sulla parte vuoto del cerchio di avanzamento
  • backgroundColorInner che definisce il colore dello sfondo del cerchietto interno
  • fillfull se true mostra la torta piena senza il cerchietto centrale
  • color che definisce il colore proprio del cerchio di avanzamento
  • height che definisce l'altezza e la larghezza del cerchio
  • size dimensioni senza specificare height ('xs' | 'sm' | 'md' | 'lg' | 'xlg')
  • showText se true mostra il testo di avanzamento percentuale
  • hideOnComplete se true nasconde la progress al termine
  • showOnStart se true visualizza la progress dopo l'avvio (necessita di un pulsante o function che fa partire il loading)
  • intervalGrow in secondi indica il tempo di avanzamento del loader
  • timeoutHide in secondi indica il tempo di spegnimento del loader
  • showDot se true mostra un pallino in movimento
  • displayInline se true mostra il loader accanto all'elemento che lo precede (default true) 
      


        ```html
          <a class="btn btn-xs btn-primary mr-2" (click)="progressCircle1.start()">Start</a>
          <a class="btn btn-xs btn-danger mr-2" (click)="progressCircle1.stop()">Stop</a>

          <ddc-init-progress-circle #progressCircle1 [cssClass]="'mt-2'" [backgroundColor]="'darkgray'" [color]="'darkblue'"
            [height]="'16px'" [intervalGrow]="0.05" [showText]="true">
          </ddc-init-progress-circle>

        ```

Il componente Loading mostra diversi modelli di loader da visualizzare oscurando la pagina e inibendo le azioni utente durante il loading.
è possibile usare il componente alternativo GenericLoadingComponent per visualizzare il loading in qualsiasi punto della pagina e non a tutto schermo con patina
  • loading: boolean deve essere true per mostrare il loading. Default false
  • color: string colore del loader. Default #000
  • typeCss: TypCssLoading tipo di loader. Default DEFAULT
    • CIRCLE
    • DUAL_RING
    • FACEBOOK
    • HEART
    • RING
    • DEFAULT
    • ELLIPSIS
    • GRID
    • HOURGLASS
    • RIPPLE
    • SPINNER
  • imageUrl: string se definito ignora il type e mette l'immagine come loader
  • imageClass: any e imageStyle: any per definire le proprietà dell'immagine se definito imageUrl
  • position: 'center' | 'left' | 'right' = 'center' definisce la posizione. Default center 
      


        ```html

          <ddc-init-loading [loading]="true" [position]="'left'" [typeCss]="'circle'"> </ddc-init-loading>

        ```

La direttiva DdcCssVarDirective consente di assegnare proprietà di stile con valore una variabile css e viene configuarto con i seguenti parametri di input
  • cssVars Array di oggetti key,value dove key è la proprietà css e value è la variabile da assegnare 
      


        ```html
          <div ddcCssVar [cssVars]="[{key:'background', value:'dark'}, {key:'color', value:'light'}]">Text for div</div>

        ```
Esempio di settaggio variabili
Text for div
La direttiva DdcTooltip consente di aggiungere un tooltip ad un elemento HTML
  • tooltip: [string | StringTranslate] testo del tooltip (è accettato anche l'html)
  • placement: ['top' | 'left' | 'bottom' | 'right'] DEFAULT top indica il posizionamento del tooltip
  • delay: [number] DEFAULT 500 indica in millisecondi il ritardo del fade di scomparsa
  • fontSize: [string] DEFAULT 14 indica la dimensione del testo
  • background: [string] DEFAULT #282a36 indica il colore di sfondo del tooltip
  • color: [string] DEFAULT #f8f8f2 colore del testo
  • textAlign: [string] DEFAULT center allineamento del testo
  • borderSize: [string] DEFAULT 2px colore del testo
  • borderColor: [string] colore del bordo. Se non passato viene settato uguale al background
  • arrowColor: [string] colore della freccia. Se non passato viene settato uguale al background 
      


        ```html
          <div ddcTooltip [cssVars]="[{key:'background', value:'dark'}, {key:'color', value:'light'}]">Text for div</div>

        ```

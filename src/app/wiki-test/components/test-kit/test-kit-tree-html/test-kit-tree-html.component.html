Modulo Tree Html

Il componente TreeHtmlComponent consente di costruire un albero di navigazione e viene configuarto con i seguenti parametri di input
  • nodes: TreeHtmlModel[] che definisce l'array dei nodi principali
  • childrens: TreeHtmlModel[] che definisce un array di nodi figli
  • initialOptions: ITreeOptions che definisce i parametri di configurazione del componente
  • flgCheckbox: boolean se true fa comparire le checkbox sui nodi
  • flgCheckboxDistinct: boolean se true gestisce distintamente i checkbox
  • fnForChildren dato un nodo con node.children=undefined e node.hasChildren=true e le propriet√† flgLoadChildrens=true e flgUseFnForChildren=true, questa funzione identifica la chiamata da fare al click del nodo per caricare in maniera asincrona i suoi figli 
      


        ```ts
          const nodeA1 = new TreeHtmlModel();
          nodeA1.id = 1;
          nodeA1.name = 'Nodo 1';
          nodeA1.hasChildren = true; /* important */
          ...
          const nodeA1_1 = new TreeHtmlModel();
          nodeA1_1.id = 1.1;
          nodeA1_1.name = 'Foglia 1.1';
          nodeA1_1.hasChildren = false; /* important */
          ...

          nodeA1.children = [nodeA1_1, nodeA1_2];

          this.nodesA = [nodeA1, nodeA2];

        ```

        ```html

          <ddc-init-tree-html [nodes]="nodesA"></ddc-init-tree-html >

        ```

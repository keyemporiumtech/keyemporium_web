Modulo Routing

La classe PageUtility consente di utilizzare delle funzioni per la gestione di url e parametri
  • cleanUrl rimuove dalla url tutto quello che precede la navigazione interna (host, cancelletto ...)
  • splitUrl scompone una url in due parti, l'array della url e l'array dei parametri con il segno =
  • splitUrlNavigationExtrasQueryParameters scompone una url generando i parametri come NavigationExtras
      


        ```ts
          PageUtility.splitUrlNavigationExtrasQueryParameters(this.router.url); // scompone la url corrente e se ci sono parametri li mette in extras


        ```

Modulo Validators

La classe MagicValidatorUtil consente di costruire un abstract control settando in automatico
  • una lista di MessageValidatorInterface per la stampa dei messaggi di validazione
  • una lista di Validatori da aggiungere al campo
  • e un valore di default
In questo modo Ã¨ possibile evitare di costruire l'oggetto che contiene i messaggi di validazione 
      
      
  
        ```typescript        
          form: FormGroup;
          validations: any = {};
          constructor(private fb: FormBuilder) {
            this.form = this.fb.group({
              campo1: new MagicValidatorUtil((this.validations.campo1 = []), 'ciao')
                .required()
                .isInteger()
                .build(),
              ...
            });          
        ```

        ```html
          <form *ngIf="form1" [formGroup]="form1">
            <div class="form-group">
              <label for="c1">Campo required e intero</label>
              <input type="text" id="c1" class="form-control" [formControl]="form.get('campo')" ddcValidation
                [validations]="validations.campo" [flgOnInitValidation]="true" />
            </div>
          </form>
        ```

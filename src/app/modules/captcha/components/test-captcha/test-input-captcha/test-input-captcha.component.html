Componente captcha
Componente per la gestione di un captcha homemade.
Parametri di Input:
  • field: FormFieldModel - Necessario per memorizzazione dei valori e validazioni
  • keyClient: string - Chiave necessaria per la generazione del token
  • automaticVerify : boolean - Se impostato a true verifica automaticamente la validità del token generato
  • label: string | StringTranslate - Label da mostrare: Di default è 'APP.LABEL.CAPTCHA.NOT_ROBOT'
  • cssClass: any - Classe css contenitore
  • cssStyle: any - Stile css contenitore
  • cssClassImage: any - Classe css immagine captcha
  • cssStyleImage: any - Stile css immagine captcha
  • borderColor: string - Colore del bordo: Default '#000'
  • imageHeight: string - Altezza immagine captcha: Default '3.5rem'
Parametri di Output:
  • generated: EventEmitter string - Emette il token generato al click
  • verified: EventEmitter boolean - Emette il risultato della verifica del token
